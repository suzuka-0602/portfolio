# portfolio

HTML と CSS で作成した自己紹介ポートフォリオのテンプレートです。ヒーロー、学習した参考書リスト、経歴タイムライン、パブリケーションの各セクションで構成しており、柔らかなライトトーンでまとめています。

## ファイル構成
- `index.html`: ページのマークアップ。ナビゲーション、ヒーロー、各セクションのコンテンツを含みます。
- `styles.css`: 全体のレイアウトや配色、カードやタイムライン、タグのスタイルを定義しています。

## 確認・チェック方法
- 静的サイトのため、ブラウザで `index.html` を開くだけで表示確認ができます。
- 任意で HTML の lint を行う場合は、以下を参考にしてください（ネットワーク制限がある環境では npm インストールが失敗することがあります）。
  ```bash
  npx htmlhint index.html
  ```

## Git でコンフリクトが発生した場合の対処メモ
1. `git status` で競合中のファイルを確認し、ファイル内の `<<<<<<<`, `=======`, `>>>>>>>` を探します。
2. 自分の変更と相手の変更を比較し、どちらかを採用するか手動で統合します。マーカーは必ず削除してください。
3. 変更後にビルドやフォーマットに問題がないか確認します。必要ならテストや lint を実行します。
4. 解消したファイルを `git add` してステージし、`git status` で `all conflicts fixed` を確認したら通常通りコミット・プッシュします。
5. PR 画面でも差分を再確認し、意図通りに反映されているか確認します。必要に応じてレビューコメントを追加してください。
